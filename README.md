# こうかとんラン

## 実行環境の必要条件
* python >= 3.10
* pygame >= 2.1

## ゲームの概要
* 横スクロールでジャンプして障害物をさけるゲーム
* 参考URL：[参考ゲーム](https://dinorunner.com/jp/)

## ゲームの遊び方
* スペースでジャンプをして障害物を避けるゲーム
* コインを取って残機が増える
* スターを取ると無敵状態になる
* ゲージが貯まると2段ジャンプになる
* 障害物に当たるとゲームオーバーになる
* 10コインで障害物を球で壊せる機能を実装
## ゲームの実装
### 共通基本機能
* 時間がたつとスピードアップする
* 障害物の設置
* ゲームオーバー画面
* リトライ機能
* コインのカウント機能
* スコアのカウント機能
### 分担追加機能
* スターを作成する　(担当: けんしん)
* しゃがみ,障害物 (担当: ほりえ)
* 残機 (担当: りお)
* 二段ジャンプ(担当:　やぎりく)
* 背景画像,キャラ画像追加,BGM(担当: かんた)
